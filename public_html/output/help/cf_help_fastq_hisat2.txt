
------------------------
FastQ to Hisat2 Pipeline
------------------------
This pipeline takes FastQ files as input, runs FastQC, Trim Galore,
then aligns with hisat2. It requires a genome reference with a
corresponding hisat2 index base in the configuration. If a GTF file
is defined for the genome idex it will be used, otherwise Hisat2 will
align without gene model annotations.

--------------------
 Pipeline:
--------------------

#fastqc
#trim_galore
	#hisat2
