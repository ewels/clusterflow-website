

------------------------
BWA to preseq
------------------------
This pipeline takes FastQ files as input, runs BWA to align, sorts
the alignments using samtools sort and indexes using samtools index.
Preseq lc_extrap is then run on the alignments and the results are
plotted individually. Finally, a summary module takes all preseq
files generated by the pipeline and plots them on a single graph.
The module requires a bwa genome reference. Preseq plotting requires
the ngi_visualizations package. See 'cf --help preseq_plot' for
more information.
