

------------------------
preseq
------------------------
This pipeline takes BAM / SAM files and, uses samtools to sort and
index the files (skipping sorting if already done).
Preseq lc_extrap is then run on the alignments and the results are
plotted individually. Finally, a summary module takes all preseq
files generated by the pipeline and plots them on a single graph.
Preseq plotting requires the ngi_visualizations package.
See 'cf --help preseq_plot' for more information.
